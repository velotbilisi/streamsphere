# 🎬 StreamSphere — by Gramito Gelenidze

StreamSphere is a cloud-native video streaming platform (YouTube clone) built using:

- Cloudflare Pages (Frontend Hosting)
- Cloudflare Workers (Backend API)
- Cloudflare D1 (Database)
- Cloudflare Workers AI (AI Assistant)

## Features

- Static YouTube video streaming
- Persistent comment system
- AI Assistant powered by LLM
- Fully serverless architecture
- CI/CD with GitHub integration

## Architecture

Frontend (Cloudflare Pages)
→ Worker API
→ D1 Database
→ Workers AI
